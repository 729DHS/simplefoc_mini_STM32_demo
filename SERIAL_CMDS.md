# SimpleFOC 串口命令参考

**串口协议**: UART1, PB6(TX) / PB7(RX), 115200 8N1  
**格式**: 单字母命令 + 可选数值，换行结束。大小写不敏感。  
**示例**: `T90⏎` → 转到 90°

---

## 一、模式切换

| 命令 | 示例 | 效果 |
|------|------|------|
| `M0` | `M0` | 开环速度模式 — 电机以目标转速匀速旋转 |
| `M1` | `M1` | 位置伺服模式 — 编码器 + PID 位置闭环 |

## 二、伺服控制

| 命令 | 示例 | 单位 | 效果 |
|------|------|------|------|
| `T<角度>` | `T90` `T-45` `T180` | 度 (°) | 设定目标角度 (伺服模式) |
| `V<0~1>` | `V0.3` `V0.15` | — | 电压限幅 (PWM 占空比上限) |
| `E1` | `E1` | — | 使能电机 (上电自动) |
| `E0` | `E0` | — | 禁用电机 (50% 占空比，无力) |
| `S<转速>` | `S60` `S200` | RPM | 设定开环转速 (速度模式) |
| `P<极对数>` | `P7` `P14` | — | 电机极对数 (2804=7, 1~100) |

## 三、PID 调参

| 命令 | 示例 | 推荐范围 (2804) | 效果 |
|------|------|-----------------|------|
| `Kp<值>` | `Kp0.05` | 0.03 ~ 0.3 | 比例增益。太大→振荡，太小→软 |
| `Ki<值>` | `Ki0.02` | 0 ~ 0.1 | 积分增益。消除静差，大→迟滞 |
| `Kd<值>` | `Kd0.05` | 0 ~ 0.5 | 微分阻尼。大→噪声抖，小→过冲 |

> **调参后自动重置积分**，不用额外操作。  
> **D 项已改为测量值微分 + 低通滤波**，setpoint 突变不会触发 D 尖峰。  
> 起步建议: `Kp0.05` + `Ki0` + `Kd0` → 慢慢加 Ki 补静差 → 加 Kd 减过冲。

## 四、诊断

| 命令 | 效果 |
|------|------|
| `D` | 编码器诊断: 磁铁检测 + 连续读 5 次原始值 (直接 I2C 读，不受缓存影响) |
| `?` | 打印完整状态: 模式、角度、误差、PID、I2C 错误计数 |
| `C` | 切换自动打印 (0.5 秒一次, 调试时可关掉) |
| `H` | 简短帮助 |

## 五、自动状态行格式

```
> A:1826 deg: 160.5 err: +0.5 tgt: 164.4 v:0.40 [SERVO] i2c:0
```

| 字段 | 含义 |
|------|------|
| `A:1826` | AS5600 原始值 (0~4095) |
| `deg:160.5` | 机械角度 (度) |
| `err:+0.5` | 位置误差 = 目标 - 实际 |
| `tgt:164.4` | 目标角度 |
| `v:0.40` | 电压限幅 |
| `[SERVO]` | 模式: SERVO / SPEED / OFF |
| `i2c:0` | I2C 连续失败计数。 >0 说明通信有问题 |

## 六、典型调试流程

```
# 1. 检查编码器
D
# 应该看到 Magnet: OK, 5 次读数连续

# 2. 关自动打印 (不刷屏)
C

# 3. 从零开始调 PID
V0.3        # 电压限制 30%
Kp0.05      # 小 P 起步
Ki0         # 关积分
Kd0         # 关微分
E1          # 使能

# 4. 测回正力
T90         # 设目标 90° → 用手转电机, 感受回正力
T180        # 设目标 180°

# 5. 调参 (根据手感)
Kp0.1       # 力不够→加 P
Ki0.02      # 有静差→加 I
Kd0.1       # 过冲大→加 D (现在不会因噪声抖了)

# 6. 看状态
?
```

## 七、故障排查

| 现象 | 检查 |
|------|------|
| `i2c:N` 持续增长 | I2C 通信故障, 检查 PB8/PB9 接线和磁铁 |
| 电机完全无力 | `?` 看 Enabled, 看 I2C 是否 0 |
| 电机一直抖 | 降 Kp (如 Kp0.03), 电压限幅降到 V0.2 |
| 电机过冲停不住 | 加 Kd (如 Kd0.1~0.3) |
| E1 后电机不动 | 看 err: 是否为 0.0 (说明已在目标位置) |
| `EMERGENCY` | I2C 失联 >0.5s, 电机自动禁能, 检查硬件 |
